# Coffee-Shop-Sales-Performance-Dashboard
An interactive Excel dashboard analyzing 50,000+ sales transactions across multiple coffee shop locations. Features automated data cleaning via Power Query, complex data modeling with Pivot Tables, and dynamic visualizations to identify peak sales hours and product performance.
## Dataset Used 
- <a href= "https://github.com/kedarjadhav365-cell/Coffee-Shop-Sales-Performance-Dashboard/blob/main/Coffee_shop_sales_analysis_project.xlsx"> Dataset </a>
## Business Question Answered 

**What are the peak transaction hours, and how should staffing be adjusted to minimize wait times?**

**Which product categories drive the highest sales volume versus the highest revenue?**

**How do sales patterns differ across the Astoria, Hell's Kitchen, and Lower Manhattan locations?**

**What is the month-over-month trend in quantity sold, and how can we optimize stock levels?**

**What is the distribution of product price points, and how does it correlate with transaction frequency?**
